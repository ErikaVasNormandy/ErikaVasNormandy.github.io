Resources Used

  • https://stackoverflow.com/questions/44212134/an-error-configuring-nginx-with-angular-2-failed-to-load-resource-neterr-con
    • Failed to load resource: net::ERR_CONTENT_LENGTH_MISMATCH
    • This only happened when loading "scavengers-repo.com", and did not occur with "scavengers-repo.com:8081", indicating an error on either the nginx side or python end
    • Post advises looking into error logs in logs/nginx_error.log; (must be enabled)
    • Turns out this is primarily a permissions issue
  • https://github.com/dockerfile/nginx/issues/4
    • How do we change the permissions in nginx then?
    • "Failed to load resource: net::ERR_CONTENT_LENGTH_MISMATCH"
    • chown -R [username]:[username] /var/lib/nginx
    • SOLVED
